# Worldwide Weather

A React Native weather application built with Expo featuring animated weather effects and a beautiful UI.

## Features

- Current weather display with automatic location detection
- 5-day weather forecast
- Animated weather backgrounds (rain, snow, clouds, sun, night, thunderstorm)
- Temperature unit toggle (Celsius/Fahrenheit)
- City search functionality
- Dynamic color themes based on weather conditions

## Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher)
- [Expo CLI](https://docs.expo.dev/get-started/installation/) or use npx

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/alessisheinman/Worldwide-weather.git
   cd Worldwide-weather
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the app:
   ```bash
   npm start
   ```

4. Run on your preferred platform:
   - Press `w` for web
   - Press `a` for Android (requires emulator or Expo Go app)
   - Press `i` for iOS (requires simulator or Expo Go app)

## Technologies

- React Native
- Expo
- OpenWeatherMap API
- React Native Reanimated

## API

This app uses the [OpenWeatherMap API](https://openweathermap.org/api). The API key is included in the repository for convenience.
